输入一行文本, 找出文本中全部的单词, 并按字典序输出, 每行只输出一个单词.

注意: 程序不区分大小写, 即 ``Apple``, ``apple``, ``Apple`` 应视为同一个单词.

例如:

.. code-block:: text
  :linenos:

  input:
    This Apple is red while that apple is GREEN

  output:
    apple
    green
    is
    red
    that
    this
    while
